# Example Session Protocol (v0.1.0)

## Objective
Exploratory check whether rank-1 hits exceed chance (25%) in a 4-candidate forced-choice task.

## Fixed design
- Trials: 40
- Candidates per trial: 4
- Hit definition: target ranked at position 1
- Primary metric: hit rate
- Exploratory test: binomial P(X >= hits | p=0.25)

## Pre-session checklist
- Confirm participant is informed and voluntary.
- No intoxication, sleep deprivation, breath manipulation, or medical risk behaviors.
- Confirm quiet environment and no external cue channels.
- Register exclusion rules before start.

## Trial procedure
1. Participant writes impression text.
2. Candidate set is revealed only after impression lock.
3. Participant ranks all 4 candidates.
4. Participant sets confidence (0-100).
