# EXPERIMENT PROTOCOL (V0.1)

1. Create a session with fixed trial count and candidate count.
2. For each trial, randomly choose 4 candidates and one actual target.
3. Participant writes impressions before ranking candidates.
4. Participant ranks all candidates from 1 to 4.
5. Score hit only when actual target is rank 1.
6. Compute session hit rate and compare to chance level.
7. Compute exploratory binomial probability (no proof claim).
8. Archive all settings and outcomes transparently.

9. Export/Import JSON ermöglicht transparente Datenspeicherung und Nachprüfung.

10. In double-blind mode, candidate options remain hidden until impression text is locked; display order is randomized per trial.

11. Each session writes an append-only audit trail of major events with ISO timestamps.
