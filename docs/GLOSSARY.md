# Glossary

- **Forced-choice task**: Experimental format where participants choose/rank among predefined options.
- **Target image**: The hidden correct image selected randomly for one trial.
- **Candidate set**: The set of images shown for ranking in a trial.
- **Rank-1 hit**: Success criterion where the real target is placed first.
- **Chance rate**: Expected hit rate under random guessing (e.g., 1/4 = 25%).
- **Binomial p-value**: Probability of observing at least the given number of hits under the chance model.
- **Wilson interval**: Confidence interval for a proportion with better small-sample behavior than Wald interval.
- **Effect size (Cohen's h)**: Standardized difference between observed hit rate and chance rate.
- **Preregistration**: Declaring planned design and exclusion rules before data collection.
- **Double-blind guard**: UI/protocol behavior that reduces information leakage before ranking.
- **Sensory leakage**: Unintended cues that can reveal target information.
- **Audit log**: Timestamped record of key session events.
- **Replication mode**: Running protocol-consistent sessions across participants/sessions.
- **Integrity hash**: SHA-256 fingerprint for tamper-evident session content checks.
- **Exploratory result**: A finding that may be interesting but is not confirmatory evidence.
