# ExperimentX

ExperimentX is an exploratory open-science tool for controlled target-identification experiments.

It is designed to test whether participants can rank hidden target images above chance under documented conditions.

ExperimentX does not prove paranormal perception, non-local consciousness, life after death, or any specific metaphysical theory. Positive results are treated as statistically interesting observations requiring replication under stricter controls.

**Deutsch:** ExperimentX ist kein Beweisgerät für Bewusstsein, Seele oder Jenseits. Es ist ein methodisches Werkzeug, um eine konkrete Frage sauberer zu testen: Liegen Treffer unter kontrollierten Bedingungen über Zufall?

## Local usage

```bash
npm install
npm run dev
npm test
```
